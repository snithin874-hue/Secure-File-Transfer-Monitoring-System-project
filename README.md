# Important Points About the Project

## Project Title

**Secure File Transfer Monitoring System**

## Project Overview

The Secure File Transfer Monitoring System is a cybersecurity-based application designed to monitor file activities in real time. It helps organizations protect sensitive information by tracking file transfers, detecting unauthorized access, verifying file integrity using cryptographic hashing, and maintaining detailed audit logs. The system improves data security by identifying suspicious file movements and generating alerts whenever security policies are violated.

---

# Project Guidelines

The following guidelines were followed during the development of the project:

* Monitor all file system activities continuously.
* Detect file creation, modification, deletion, and movement events.
* Identify sensitive files using predefined keywords and file extensions.
* Generate SHA-256 hash values for uploaded files.
* Verify file integrity before and after file transfers.
* Record all user activities in audit logs with timestamps.
* Generate security alerts for unauthorized access or suspicious file movement.
* Produce audit reports that summarize all monitored events.
* Ensure data confidentiality, integrity, and availability throughout the monitoring process.
* Maintain an easy-to-use dashboard for administrators.
